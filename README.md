Code is run on a Raspberry Pi Pico 2W using Micropython.

To develop:
1. Install `paulober.pico-w-go` vscode extension
2. Plug in via USB
3. If prompted, install Micropython firmware
4. Use the "Run" command to get an interactive prompt

To deploy:
1. Hold down the BOOTSEL button while plugging in USB. Keep held until USB Mass Storage device appears
2. Copy "main.py" onto the device
